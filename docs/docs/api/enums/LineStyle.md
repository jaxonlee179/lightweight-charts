## Enumeration members

### Dashed

• **Dashed** = `2`

A dashed line.

___

### Dotted

• **Dotted** = `1`

A dotted line.

___

### LargeDashed

• **LargeDashed** = `3`

A dashed line with bigger dashes.

___

### Solid

• **Solid** = `0`

A solid line.

___

### SparseDotted

• **SparseDotted** = `4`

A dottled line with more space between dots.
