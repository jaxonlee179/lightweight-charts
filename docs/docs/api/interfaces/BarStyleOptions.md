Represents style options for a bar series.

## Properties

### downColor

• **downColor**: `string`

Color of falling bars.

___

### openVisible

• **openVisible**: `boolean`

Show open lines on bars.

___

### thinBars

• **thinBars**: `boolean`

Show bars as sticks.

___

### upColor

• **upColor**: `string`

Color of rising bars.
