___

• **lineStyle**: [`LineStyle`](../enums/LineStyle)

___

### lineWidth

• **lineWidth**: [`LineWidth`](../#linewidth)

Price line's width in pixels.

___

### price

• **price**: `number`

Price line's value.

___

### title

• **title**: `string`
