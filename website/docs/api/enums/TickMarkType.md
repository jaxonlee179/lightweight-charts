Represents the type of a tick mark on the time axis.

## Enumeration members

### DayOfMonth

• **DayOfMonth** = `2`

A day of the month.

___

### Month

• **Month** = `1`

The start of the month (e.g. it's the first tick mark in a month).

___

### Time

• **Time** = `3`

A time without seconds.

___

### TimeWithSeconds

• **TimeWithSeconds** = `4`

A time with seconds.

___

### Year

• **Year** = `0`

The start of the year (e.g. it's the first tick mark in a year).
