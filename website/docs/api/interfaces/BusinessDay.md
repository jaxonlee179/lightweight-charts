Represents a time as a day/week/month.

```js
const day = { year: 2019, month: 6, day: 1 }; // June 1, 2019
```

## Properties

### day

• **day**: `number`

The day.

___

### month

• **month**: `number`

The month.

___

### year

• **year**: `number`

The year.
